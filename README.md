# 📱 HERO.IO

A modern app discovery platform where users can explore, review, and install their favorite apps.

## 📝 Description

Hero.io is a React-based web application that allows users to browse apps, view detailed information including ratings, reviews, and download stats, and install apps directly from the platform. Installed apps are persisted using localStorage so they remain available across sessions.

## 🛠 Technologies

- **React** — UI library
- **React Router** — client-side routing
- **Tailwind CSS** — utility-first styling
- **Recharts** — data visualization for ratings chart
- **FontAwesome** — icons
- **LocalStorage** — persisting installed apps
